# Daily Quote Generator

This is a simple web app that displays a random daily quote. The project is built using HTML, CSS, and JavaScript.

## Features

- Displays a random quote after clicking on "Get Your Quote!" button.
- Generated quote stored in browser.
- Only allows 1 quote per day.
